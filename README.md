# Build-a-Survey-Form

Formulario web simple para inscripción a clases de buceo.

## Descripción

Este proyecto contiene un formulario de inscripción que recopila información básica de los alumnos interesados en clases de buceo. El formulario incluye:

- Nombre
- Correo electrónico
- Edad
- Expectativas del curso
- Género
- Intereses
- Comentarios

## Uso

1. Abre `index.html` en tu navegador.
2. Completa los campos del formulario.
3. Presiona el botón `Enviar`.

## Archivos

- `index.html` — Página principal con el formulario y estilos básicos.
- `README.md` — Documentación del proyecto.

## Notas

- El formulario es estático y no guarda los datos por defecto.
- Puedes mejorar el proyecto agregando:
  - validación adicional con JavaScript,
  - envío de datos a un servidor,
  - estilos CSS externos,
  - diseño responsive.
